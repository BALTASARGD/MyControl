# **App Name**: MiControl

## Core Features:

- Income Recording: Record income entries, categorizing each by source and date.
- Expense Tracking: Track expenses with categorization for detailed analysis.
- Interactive Charts: Generate interactive charts visualizing spending patterns and trends over time.
- Monthly Savings Summary: Provide a summary of monthly savings.
- Budget Alerts: Dashboard alerts for budget thresholds. LLM is used as a tool to interpret trends in spending so as to know when to raise the alarm.
- Responsive Design: Fully responsive design for accessibility on various devices.
- Data Sync & Export: Synchronize data across devices and export reports.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to convey trust and stability in financial management.
- Background color: Very light Lavender (#E8EAF6), near white, providing a calm, neutral backdrop.
- Accent color: Teal (#009688) to highlight key actions and data points, drawing user attention.
- Body and headline font: 'Inter', a sans-serif font that offers a modern and neutral appearance, ideal for both headers and body text, ensuring readability and clarity.
- Use clean, minimalist icons for categories and actions.
- Dashboard layout with clear sections for statistics and alerts.
- Subtle transitions and animations for enhanced user experience.
